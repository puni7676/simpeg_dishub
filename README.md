# SIMPEG Dinas Perhubungan
Sistem Informasi Manajemen Kepegawaian (SIMPEG) berbasis web yang dibuat menggunakan PHP Native dan MySQL.

## Fitur
- Login Multi Role
- Dashboard
- Data Pegawai
- Absensi
- Pengajuan Cuti
- Persetujuan Cuti
- Laporan
- Notifikasi
- Pengaturan Profil
- Upload Foto Profil
- Ubah Password

## Teknologi
- PHP Native
- MySQL
- HTML
- CSS
- JavaScript
- Font Awesome

## Cara Menjalankan
1. Import database `database/simpeg.sql`
2. Simpan project pada folder `htdocs`
3. Jalankan Apache dan MySQL menggunakan XAMPP
4. Akses:

http://localhost/simpeg_dishub
